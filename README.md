### Environment Variables

the project needs to have
VUE_APP_URL=url_of_dev_backend
VUE_APP_HOST_URL=url_of_where_this_app_is_hosted

i.e.
VUE_APP_URL=http://localhost:5000
VUE_APP_HOST_URL=http://localhost:8080
